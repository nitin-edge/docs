

# Introduction to Taxonomy API

Welcome to the Taxonomy API documentation. This API provides a powerful and flexible way to manage taxonomies, functions, skill categories, and skills within your organization or application.

## What is the Taxonomy API?

The Taxonomy API is a RESTful service that allows you to create, read, update, and delete various elements of a skill-based taxonomy. It's designed to help you organize and categorize skills and functions in a structured manner.

## Key Features

- **Functions Management**: Create and manage high-level job functions.
- **Skill Categories**: Organize skills into broader categories.
- **Skills Management**: Add, update, and categorize individual skills.
- **Taxonomy Creation**: Build hierarchical relationships between functions, skill categories, and skills.
- **Search Capabilities**: Powerful search functionality across all entities.
- **Metadata Support**: Add and manage additional information like skill trends, automation probability, and more.

## Who is it for?

The Taxonomy API is ideal for:

- HR Technology Providers
- Learning and Development Platforms
- Job Boards and Recruitment Platforms
- Workforce Analytics Tools
- Any application dealing with skills and job functions
